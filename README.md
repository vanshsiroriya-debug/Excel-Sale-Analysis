# Product-Wise Sales Analysis Project 📊

## Project Overview
This project focuses on analyzing sales data to understand product performance and revenue trends. I developed this using **Microsoft Excel 2007**, demonstrating how core analytical tools can be used to derive meaningful business insights from raw data.

## Key Features & Workflow
* **Data Management:** Organized raw sales records including Order ID, Date, Customer Details, and Category.
* **Data Analysis:** Utilized **Pivot Tables** to summarize total sales figures for different product categories.
* **Data Visualization:** Created a **Product-Wise Sales Chart** to visually represent the performance of each item.

## Tools Used
* **Microsoft Excel 2007** (Pivot Tables, Charts, Data Formatting)

## Business Insights
* **Top Performing Product:** **Laptops** emerged as the highest-selling category with total sales of **₹205,000**.
* **Overall Revenue:** The total sales generated across all categories amounted to **₹590,250**.
* **Sales Distribution:** Phones and Tablets followed Laptops, showing a clear demand for electronics in the dataset.
